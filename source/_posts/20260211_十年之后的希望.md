# 十年之后的希望--《历史的拐点：2025-2035国际格局和秩序》

这本书读起来很轻松，并没有太多难懂晦涩的表述，更多的是一种逻辑上的推理和展望，关于经济类的数据我也用龙虾去抓取整理并分析，简单的小任务还是能看出不同大模型的能力参差。

作者预测，下一个十年，中美两个大国的差距有可能会缩小一点，但是在世界经济总产量的占比，各自都会增加，其他国家比重会下降；由于民粹主义和逆全球化的兴起，美国所谓的讲规则和秩序的形象会慢慢消失，各个国家都会从保护自身的角度更谨慎地和美国交往；失去法治转向人治的国际秩序有可能到达顶峰，世界各国可能会寻求一种新的讲究道义、权责一致的新秩序。

## 信息技术是大国博弈的主力

作者认为接下来十年，**数字技术与网络空间决定国力**。

• **技术决定论：** 在数字时代，技术创新（特别是AI）是国力增长的核心。2035年的大国实力对比将主要取决于数字技术创新能力和网络空间实力。

• **双重空间竞争：** 大国博弈将同时在自然空间和网络空间进行，且网络空间的重要性日益上升。到2035年，网络空间的国际秩序对整体国际关系的影响可能超过自然空间。一个国家若不能确保网络空间的安全，就无法保卫其在自然空间的领土和主权。

• **技术标准分裂：** 中美竞争可能导致全球形成两套不同的数字技术标准体系，其他国家将面临选边站或并行使用的抉择。

作者给出了自己的建议：

- 在数字时代，**技术创新**为大国战略竞争的核心，要吸引世界一流创新人才，缩短AI技术和美国的差距和提升网络空间防护能力；
- **经济逆全球化**使全球市场碎片化，放宽外汇管制更大力度吸引外资，助力中国标准走向世界，加大世界范围内的军事合作；
- **防范民粹主义冲突，扩大统一战线**，强调互补互利**以国家利益界定敌友，淡化身份差别**，**防范国内民粹主义**，保证高质量对外开放目标的实现。

## 技术创新的领域

作者提到

>  今后十年，**人工智能**、**量子科技**、**人形机器人**、**低空飞行器**、**自动驾驶**、**3D建筑打印**等技术将可能逐渐成为广泛应用的技术，大国可能都将科技创新视为促进经济发展的核心要素。在逆全球化的趋势下，大国决策者们很可能将科技保密视为经济安全的核心。

这次的读书笔记对书中的观点做了整理和摘录，目的有几个：

- 十年后，自己的孩子将走向社会，有条件的话做父母的可以帮忙看看未来的趋势会是什么样？
- 十年间，民粹主义导致的“黑天鹅”事件一定会增加，对未来几年不能过于乐观，做好工作和生活的应对。
- 十年内，自己从事的技术工作仍然是产业革命、经济发展和社会治理的重要部分，技术投入非常值得。

这几个问题，自己能找到答案，尤其是从权威的、负责任的专家处，得到一些认同和启发，这是我最大的收获。那么接下来，书中关于创新技术和网络安全的发展，我的看法会和书中的不一样，毕竟作者是国际关系的专家，我是这个信息行业的一份子，技术发展的弄潮儿，对技术和发展会有自己独特的看法。

# 我眼中的图景

AI的发展每个月仿佛都在更新换代，一个个所谓的不可能的任务，正在逐渐的被攻克，人类曾经花了几千年才建立的科学大厦，AI可能在很短的时间内就能掌握。**人工智能，正在极大地影响着科研工作和日常生活**。

## 科研

以Deepmind为代表的AI4Science领域：

| **系统名称** | **解决的问题** | **核心意义** |
| --- | --- | --- |
| **AlphaFold 3** | 蛋白质、配体、核酸的**三维结构** | 决定了“物质的长相”，加速药物筛选。 |
| **AlphaGenome** | **基因调控与变异预测** | 决定了“细胞如何运行”，破解遗传病根源。 |
| **AlphaGeometry** | **数学逻辑推理** | 赋予 AI 严密的演绎推理能力，不再“胡言乱语”。 |
| **AlphaDev** | **底层算法优化** | 让 AI 自己写出性能更强的 C++ 排序和散列算法。 |
| **AlphaQubit** | **量子纠错** | 解决物理硬件瓶颈，让量子计算走向实用 |

随着 AlphaGenome 的加入，DeepMind 的 AI 科学版图已经变得非常清晰，**AlphaGenome 的出现意味着我们进入了“可编程生物学”时代。** 它像是一个编译器，把看不懂的 ATCG 代码翻译成人类能理解的生理功能和致病机理。2024 年拿了诺奖的 AlphaFold 只是开端，AlphaGenome 才是真正深入生命底层逻辑的利器。

## 应用

在日常生活中，以Anthropic和OpenAI为代表的企业：

| **领域** | **Anthropic / OpenAI 代表进展** | **解决问题 + 核心意义（合并）** |
| --- | --- | --- |
| 对话 | Claude / ChatGPT：更稳的指令遵循、更多模态、更长上下文 | 把“只能问答的聊天”变成“能持续协作的助手”：材料多也不易丢，任务长也能按目标推进 |
| 应用 | 工作台化交付（Claude 的产物沉淀）+ 桌面化与应用生态（ChatGPT/GPTs 等） | 把“散落在聊天记录里的答案”变成“可交付、可复用、可协作的工作成果与流程” |
| 智能体 | Computer Use / Operator 等“能操作电脑与网页”的代理 | 解决“会说不会做”：让 AI 直接完成多步流程，从建议升级为代办执行 |
| MCP（skills） | MCP 标准（Anthropic 发起，OpenAI 生态支持） | 解决工具/数据接入碎片化：把外部系统统一成可插拔 **skills**，让智能体能力可迁移、可组合、可治理 |

Anthropic 和 OpenAI 的进步不只是“模型更聪明”，而是把 AI 从“会聊天”推到“会干活”，再用 **MCP 这种 skills 接口**把能力接进真实世界的工具与数据里——智能体于是从单点能力，变成可规模化的执行系统。

## 软件行业

| 比较维度 | **OpenAI / Anthropic（主流路线）** | **OpenClaw（异端路线）** | **“哲学差异”** |
| --- | --- | --- | --- |
| **多模型能力调用** | 模型像“电厂”，你用它供电 | 模型像“插座”，你自己选电源（多家模型可路由） | 从“相信一家”变成“调度众生” |
| **个人系统主权（自定义/扩展/主权合一）** | 你在产品里配置，主权在平台边界内 | 你把它装进自己的系统里改造，主权更靠近个人 | 从“租用工具”变成“拥有工具” |
| **执行与现实连接（智能体）** | 能做事但更克制，护栏更重 | 更像本机管家，更敢接触文件/命令/流程 | 从“建议”走向“托付” |
| **治理与可托付性（MCP/信任链/行动账本合一）** | 倾向平台化治理：标准接口、权限、审计、评估、在环审批 | 扩展更自由，也更需要标准化接口与可审计、可回滚的“文明化”治理 | 从“能力竞争”变成“责任竞争” |
| **软件形态与产业重写（工程模式/行业挑战合一）** | 软件变成“产品壳 + agent 编排”，平台重建秩序 | 软件变成“个人流程/skills 集合”，边界更易被打散 | 不是代码死了，是“菜单式软件”在死 |

OpenClaw 更像“新个人主义”——把智能做成个人系统：多模型只是燃料切换，skills 才是可更换的器官，而 **MCP 是把器官工业化、可审计化的神经接口**。真正的分水岭在三件事：**主权归属（钥匙在谁手里）、信任链是否成立（扩展如何不被投毒）、行动是否可复盘（能否回放与追责）**。
